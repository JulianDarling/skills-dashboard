    30 秒学会用 Skill

    你只需要说话

    不用记命令。直接跟 Claude 描述你要做的事，它会自动找到对应的 skill 并执行。比如你说"帮我设计个调研方案"，它就知道该用 /research-plan。

    每张卡片下面的灰色标签就是触发词——你说出其中任何一个词，skill 就会启动。

    想精确控制时

    输入 / 加 skill 名称可以强制触发。适合你明确知道要用哪个、不想等 Claude 猜的时候。

    两个 skill 可以接力：先 /research-plan 出方案，再 /persuasion-proposal 包装成提案给老板。前一个的产出就是后一个的输入。

    觉得某件事你总在重复做？

    那就该把它变成 skill。跟 Claude 说"帮我创建一个 skill"或者直接 /skill-creator，描述你每次重复的工作流程，它会帮你生成一个可复用的 skill 文件。

    下次再遇到同样的事，一句话就触发整套流程。这就是 skill 的价值——把经验固化成自动化。
